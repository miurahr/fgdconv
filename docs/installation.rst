Installation
============

Install from wheel
------------------

It is recommend to use with virtual environment for POSIX system (e.g. Linux, MacOSX).

HOW TO INSTALL::

    $ virtualenv --python=python3.5 venv
    $ . venv/bin/activate
    $ pip install -r requirement.txt
    $ pip wheel .
    $ pip install jpgisgml2gml-0.0.1-py2-none-any.wheel
